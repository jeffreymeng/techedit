Login Methods

When you are about to login to techedit, you have the option of logging in using a techedit account, or logging in using a Google or Github account. While you need to create an account in order to login using a techedit account, the first time you login with a Google or Github account your account will be automatically created. You also have the option of adding login methods to your account. To do this, login to techedit using a login method you already use(If you have mutiple, you will need to delete them so there is only one left). Then, go to the account settings page(in the navagation bar), and click on the Login Methods tab.
